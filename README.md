# Project-Y

## Descripcion

El juego sera un bullet hell con mecánicas de roguelike, donde con cada mejora que el jugador eliga sera una mejora para el jugador y para el enemigo teniendo que elegir no solo algo que te beneficie, sino que no beneficie demasiado al enemigo.

El jugador al matar/derrotar enemigos pequeños conseguira puntos/energia y al conseguir una determinada cantidad el jugador tendrá una elección de mejoras (seguramente entre 3 opciones), cada mejora a parte de mejorar al jugador también mejorará a los jefes. Algo como por ejemplo obtienes x2 de velocidad y el enemigo hace x1.5 mas de daño o algo asi.

También en la seleccion de mejora habia pensado en que también hubiera algo como especiales, un ataque fuerte que requiere energía, el cual obtienes en una selección de mejoras, cuando ya tenga uno y vuelves a elegir otro especial el que tengas se sustituirá por el nuevo.
